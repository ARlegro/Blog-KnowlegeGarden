dp 배열 만들 때 개수를 어케 정하지? 하다가 그냥 dict쓰면 편할 것 같아서 dict쓰니까 깔!끔
```python
import sys
from collections import deque, defaultdict
input = sys.stdin.readline


def solution(coin_type: list, target_num: int):
  # idx = 합, value = 최소 개수  
  dp_dict = defaultdict(int)
  queue = deque([(0,0)])
  # 종료 조건 : q가 비거나 target_num이 되거나
  while queue:
    count, cur_sum = queue.popleft()
    for c in coin_type:
      add_sum = cur_sum + c
      if add_sum == target_num:
        return count + 1

      if dp_dict[cur_sum + c] != 0 or add_sum > target_num:
        continue

      dp_dict[add_sum] = count + 1
      queue.append((count+1, add_sum))

  return -1

if __name__ == "__main__":
  n, k = map(int, input().split())
  coin_type = [int(input()) for _ in range(n)]
  coin_type.sort()
  print(solution(coin_type, k))
```
