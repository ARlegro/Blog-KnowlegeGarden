> [!TIP] Python에서는 자바처럼 매개변수나 반환값에 **명시적인 타입 선언이 필수**는 아니지만, Python 3.5+부터는 **타입 힌트(type hint)** 를 사용할 수 있다

### 0.1.  기본 변수 타입 힌트 

```python
name: str = "Yohan"
age: int = 36
height: float = 177.5
is_active: bool = True
```

### 0.2.  함수에 타입 힌트 

```PYTHON

from typing import List, Dict, Any

def get_random_user_pipe(users: List[str]) -> List[Dict[str, Any]]:
    return [
        {
            "$match": {
                "_id": {"$nin": users}
            }
        },
        {
            "$sample": {
                "size": 1
            }
        }
    ]
```
users 매개변수는 `List<String>`형태 
return Value는 `List[Dict[str, Any]]` 형태 




### 0.3.  번외 : typing 주요 타입

| 타입 힌트         | 설명                      |
| ------------- | ----------------------- |
| `List[T]`     | T 타입 요소들로 구성된 리스트       |
| `Dict[K, V]`  | K, V 타입의 키-값 쌍을 가진 딕셔너리 |
| `Optional[T]` | T 또는 `None` 가능          |
| `Union[A, B]` | A 또는 B 중 하나 가능          |
| `Any`         | 어떤 타입이든 가능 (최후의 수단)     |
